# `quantum_dot/metrics` - Metrics module

This module contains the metrics used in the diagnostics class of the Quantum Dot.

## What is here
- `bell.py` - Simple Bell fidelity analyzer
- `decompose.py` - Provides population decomposition (p0, p1_total, p2_exact)
- `entanglement.py` - Provides a logarithmic negativity for entanglement presence testing
- `linops.py` - linear operations helper
- `mode_registry.py` - Data container
- `overlap.py` - Lorentzian spectral overlap and HOM visibility
- `photon_count.py` - Expected photon numbers on early/late emissions
- `registy.py` - Data container
- `two_photon.py` - Exactly one photon in early and exactly one photon in late emission plus post selection
